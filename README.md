# Psikolog Web Sitesi

Bu proje, HTML ve CSS kullanılarak hazırlanmış sade ve huzur veren bir psikolog web sitesidir. 
Amaç, siteye giren kişilerin kendini rahat ve güvende hissetmesini sağlamaktır.

## Özellikler
- Hakkımda bölümü
- Hizmetlerimiz bölümü
- İletişim formu
- Sade ve huzur veren tasarım
- Responsive yapı

## Kullanılan Teknolojiler
- HTML5
- CSS3

## Dosya Yapısı
├── index.html
├── style.css
└── images
    ├── bg.jpg
    └── logo.png
## Nasıl Çalıştırılır
1. Dosyaları bilgisayara indirin.
2. `index.html` dosyasını açın.
3. Site tarayıcıda çalışacaktır.
